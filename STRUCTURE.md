## Key Files Explained

### Extension Configuration

- **package.json**: Defines the extension metadata, activation events (Ruby, ERB, JS, JSX, TS, TSX), and snippet contributions
- **tsconfig.json**: TypeScript compiler settings targeting ES2020 with strict mode
- **.eslintrc.json**: Code quality and style rules

### Source Code

- **src/extension.ts**: Main extension logic with:
  - `activate()` function that runs when extension loads
  - Console logging for debugging
  - TODO comments for autocomplete and hover providers
  - `deactivate()` cleanup function

### Snippets

- **snippets/rails.json**: 7 snippets for Rails/ERB (pb_button, pb_card, pb_flex, etc.)
- **snippets/react.json**: 8 snippets for React (pbButton, pbCard, pbImport, etc.)

### Data

- **data/playbook.json**: Structured metadata for 7 Playbook components:
  - Button, Card, Flex, Body, Title, Avatar, Icon
  - Includes Rails/React mappings, descriptions, and prop schemas
  - Not yet wired into extension logic (future-facing)

### Development

- **.vscode/launch.json**: Two configurations:
  - "Run Extension" - Launch extension in debug mode (F5)
  - "Extension Tests" - Run test suite (not yet implemented)
